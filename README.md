# Worldbuilding of Spiral Knight

This repository contains the sources of a study of Spiral Knights fictional setting, lore, mythology, and its worldbuilding -- [`src/lib/burning_stars.mdx`](./src/lib/burning_stars.md), -- and its additional materials.

The document is built for multiple targets: `.pdf` or a webpage:

Build requirements:

- `Node` or other compatible javascript runtime
- `npm`-compatible package manager

1. Install the dependencies:

  ```sh
  npm install
  ```

2. Refer to [taskfile](./taskfile.yml) for further available actions

For pdf generation (debian-only) install additional packages:

```sh
apt install weasyprint wkhtmltopdf
```
